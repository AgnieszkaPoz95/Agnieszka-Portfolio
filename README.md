## Projects
 
### 1. 🧹 World Layoffs – Data Cleaning in SQL
**Tools:** MySQL · MySQL Workbench  
Cleaned and standardized a real-world dataset of 2,000+ global tech layoffs (2020–2023).  
 
Key techniques used:
- Removed duplicates using `ROW_NUMBER()` with `PARTITION BY`
- Standardized inconsistent values (industry names, country formatting)
- Populated NULL values using a self-JOIN
- Converted date column from TEXT to DATE using `STR_TO_DATE()`
